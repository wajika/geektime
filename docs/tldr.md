# TL;DR;

安装 geektime-dl

```bash
mkdir geektime $$ cd geektime
python3 -m venv venv3 && source venv3/bin/activate
pip install -U geektime_dl
```



查看课程列表，获取课程 ID

```bash
geektime query
```

制作 Kindle 电子书

```bash
geektime ebook 49
```


打开电子书

```bash
open ebook/朱赟的技术管理课\[更新完毕\].mobi
```

这就足够了。

如果您还想了解更多用法，可以进行[下一步](/intro.html)阅读。
